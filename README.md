# Personalized AI Learning Assistant

## 项目简介
本项目是一个基于智能体（Agent）和 AI 技术的个性化学习辅导平台，旨在通过数据分析和机器学习技术，帮助学生提供量身定制的学习辅导和支持。智能体能够根据学生的学习进度和需求，提供动态的辅导方案和练习，从而提升学生的学习效果。

## 功能特点
- **个性化学习推荐**：根据学生的学习进度和难点，推荐定制化的学习资源和练习。
- **智能辅导助手**：实时回答学生的问题，并根据情况提供深入的讲解和反馈。
- **数据驱动的学习分析**：利用学生的学习数据，分析其弱点和进步，自动调整辅导策略。
- **多平台支持**：支持Web、移动端等多个平台，方便学生随时随地学习。

## 技术栈
- **AI/Agent 技术**：使用 OpenAI GPT DeepSeek系列模型和自定义智能体来处理学习内容生成和问题解答。
- **前端框架**：React.js 用于开发用户界面。
- **后端技术**：Python + Flask 用于服务器端 API 服务。
- **数据库**：MySQL 用于存储学生的学习进度和数据。
